题意

有n个人，两种节目，每个人都要且只能表演一种节目，每种节目至少要一个人来演。用xix_ixi​和yiy_iyi​表示第i(1≤i≤n)i(1 \leq i \leq n)i(1≤i≤n)个人的表演两种节目的能力值。要求表演第一种节目的人中的能力最大值和表演第二种节目的能力最大值之差最小，求该最小值。

题解

贪心策略

维护两个集合s1s_1s1​和s2s_2s2​。（这两个集合都只存储y）

按xxx从大到小枚举。相当于我们取当前xix_ixi​为第一种节目最大值，大于xix_ixi​的部分自然全部扔到s2s_2s2​集合作为表演第二种节目的人，对于这一部分，我们找到yzy_zyz​最大的人zzz。对于xj≤xix_j \leq x_ixj​≤xi​的部分，我们挑选yjy_jyj​与xix_ixi​最接近的第jjj个人jjj。ans=min(ans,abs(xi−max(yj,yz)))ans = min(ans,abs(x_i-max(y_j,y_z)))ans=min(ans,abs(xi​−max(yj​,yz​)))，维护最小ans即可

贪心策略证明

由于这种贪心策略枚举了每个xix_ixi​作为答案，因此如果该贪心策略错误，意味着在某个正确的xix_ixi​下，该贪心策略选择了错误的yjy_jyj​。但是集合s1s_1s1​保证了能够取到xix_ixi​作为答案，因此对于上述的yzy_zyz​选择是唯一的。对于s2s_2s2​，我们选择了最接近xix_ixi​的yjy_jyj​作为答案，也符合最优解的性质，因此不可能有其他的最优解。因此贪心策略正确。
